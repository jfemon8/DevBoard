Question-1: What are the different ways to select an element in the DOM?

There are multiple ways to select elements in the DOM including: getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll(), getElementsByName(), childNodes, children, closest(), matches(), document.forms, document.images, document.links, etc.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML gets or sets the HTML content inside an element. It can contain HTML tags. innerText gets or sets the visible text content inside an element. It ignores hidden elements and renders the text as the user sees it. textContent gets or sets the plain text content inside an element, ignoring any HTML tags. It includes hidden text too.

Question-3: What is event delegation in the DOM?

Event Delegation is a technique in JavaScript where we attach a single event listener to a parent element instead of individual child elements. This allows events to be handled efficiently, especially when dealing with dynamic or multiple child elements. Event delegation helps manage events on dynamically generated elements and improves performance by minimizing the number of event listeners. It takes advantage of event bubbling.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a behavior in the DOM where an event triggered on an element bubbles up through its ancestors in the DOM tree, triggering the same event on each parent element, starting from the target element and up to the root of the document. Event Bubbling is the process where an event triggers an element and then propagates up to its ancestors. It allows parent elements to listen for events on their child elements, which is especially useful in event delegation.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create a new element using the document.createElement() method. We can add the created element to the DOM using methods like appendChild(), insertBefore(), or append(). We can remove an element using the removeChild() or remove() method.
